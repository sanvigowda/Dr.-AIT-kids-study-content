# Mini Project: Business Intelligence Case Study

> **Goal**: Play the role of a Junior Data Analyst for 'SuperMart'. Your task is to analyze the last 6 months of data and provide 3 actionable recommendations to the CEO.

---

## 1. The Context (What & Why)
- **What**: This project combines SQL querying and Pandas analysis to solve a real retail problem.
- **Why**: In a real job, you won't be told "run a mean function." You'll be told "we are losing money in Mumbai, find out why."
- **Where**: This type of "Deep Dive" analysis is done in every retail and e-commerce company (e.g., Reliance Retail, Amazon, Flipkart).

---

## 2. The Problem Statement
SuperMart has noticed that while total sales are high, profits are dipping. They suspect:
1. Some cities are underperforming.
2. Certain categories have very high return rates.
3. Premium members aren't visiting often enough.

---

## 3. Your Tasks (The How)

### Phase 1: SQL Extraction
Use SQL to pull a "City-wise Performance Report" showing:
- Total Sales
- Average Order Value (AOV)
- Total Transaction Count

### Phase 2: Python Analysis
Use Pandas to:
- Find the **7-day moving average** of sales to see when the dips happen.
- Identify the "Dormant Champions" (Members with high spend but no visits in 30 days).

### Phase 3: The Recommendation
Write 3 one-sentence recommendations using the **Minto Pyramid** style (Conclusion first).

---
