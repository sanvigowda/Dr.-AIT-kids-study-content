# SQL Fundamentals for Data Analytics

SQL (Structured Query Language) is the most critical tool for extracting data from databases.

## 1. SELECT and FROM
Picking columns from a table.
```sql
SELECT column1, column2
FROM table_name;
```

## 2. WHERE Clause
Filtering data based on conditions.
```sql
SELECT * FROM students
WHERE marks > 75;
```

## 3. ORDER BY
Sorting your results.
```sql
SELECT name, marks FROM students
ORDER BY marks DESC;
```
