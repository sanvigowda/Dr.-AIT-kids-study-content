##  Business Problem

**SuperMart** is a fictional retail chain with 5 stores across India. The sales manager has a dataset of transactions but doesn't know:

1. Which products are the biggest revenue drivers?
2. Are there data quality issues hiding in the raw data?
3. Which store performs best per quarter?
4. Do members spend more than walk-in customers?
5. What's the relationship between discounts and revenue?

**Your job:** Build a complete analytics pipeline that answers these questions.

##  Hints & Guidance

<details>
<summary>Hint 1: Data Cleaning Checklist</summary>

- Check `df.isnull().sum()` first
- Check `df.duplicated(subset=['transaction_id']).sum()`
- Check `df['price'].describe()` for impossible values
- Use `pd.to_datetime(df['date'], infer_datetime_format=True)` for mixed dates
- Use `.str.title()` to standardize text fields

</details>

<details>
<summary>Hint 2: Revenue Calculation</summary>

```python
df['revenue'] = df['price'] * df['quantity'] * (1 - df['discount_pct']/100)
```

</details>

<details>
<summary>Hint 3: Key GroupBy Operations</summary>

```python
# Revenue by store
df.groupby('store')['revenue'].sum()

# Multiple metrics by category
df.groupby('category').agg(
    revenue=('revenue', 'sum'),
    transactions=('transaction_id', 'count'),
    avg_basket=('revenue', 'mean')
)
```

</details>
